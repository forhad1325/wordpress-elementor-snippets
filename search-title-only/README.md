# Elementor Search — Title Only

Restricts the Elementor Search Widget to search WooCommerce product titles only, ordered by price high to low.

## Usage

1. Paste `index.php` code into your theme's `functions.php` or a custom plugin.
2. In Elementor, set your Search widget Query ID to `only_title_search_product`.

## Requirements

- Elementor Pro
- WooCommerce
