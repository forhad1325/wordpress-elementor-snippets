# WordPress Elementor Snippets

A curated collection of custom Elementor snippets, hacks, and extensions by [TrustyCoder](https://trustycoder.com).

Each snippet lives in its own subfolder with a dedicated README explaining usage.

## Snippets

| Folder | Description |
|--------|-------------|
| [tab-hover](./tab-hover/) | Activate Elementor Nested Tabs on hover |
| [tab-slider](./tab-slider/) | Autoplay/slider for Elementor Nested Tabs with progress bar |
| [search-title-only](./search-title-only/) | Restrict Elementor search widget to post titles only |
| [blog-post-search-only](./blog-post-search-only/) | Restrict Elementor search template to blog posts only |
| [accordion-first-item-collapse](./accordion-first-item-collapse/) | Collapse the first Elementor accordion item by default |
| [horizontal-scroll](./horizontal-scroll/) | Mousewheel-driven horizontal scroll for Elementor containers |
| [avoid-multiple-popup](./avoid-multiple-popup/) | Prevent multiple Elementor popups from opening simultaneously |
| [post-read-time](./post-read-time/) | Add a Post Read Time dynamic tag to Elementor |
| [author-meta-dynamic-tag](./author-meta-dynamic-tag/) | Add an Author Meta Link dynamic tag to Elementor |

## Requirements

- WordPress 5.0+
- Elementor (free) or Elementor Pro where noted
