# Elementor Horizontal Scroll

Converts an Elementor container into a horizontal scroll section driven by the mouse wheel / vertical page scroll.

## Usage

1. Add CSS classes to your Elementor containers:
   - Outer wrapper container: `horizontalScrollWrapper`
   - Inner scrolling container: `horizontalScrollTranslate`
2. Enqueue `main.js` and `style.css`.

## Requirements

- Elementor (free or Pro)
