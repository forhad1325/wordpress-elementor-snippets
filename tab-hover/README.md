# Elementor Tab Hover

Activates Elementor Nested Tabs on mouse hover instead of click. Includes a version that auto-activates the first tab on load.

## Usage

1. Add the CSS class `hoverTabs` to your Elementor Nested Tabs widget.
2. Enqueue or paste `elementor-tab-hover.js` via Elementor Custom Code or your theme.

## Requirements

- Elementor (free or Pro)
- jQuery
