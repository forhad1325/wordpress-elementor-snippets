# Elementor Tab Slider (Autoplay)

Extends Elementor Nested Tabs to autoplay like a slider with an animated progress bar. Supports vertical and horizontal layouts.

## Usage

1. Add CSS class `autoplay-tab` to your Nested Tabs widget.
2. For vertical layout also add `vertical-tab`; for horizontal add `horizontal-tab`.
3. For mobile autoplay add `mobile-autoplay`.
4. Enqueue `tab-slider.js` and `tab-slider.css`.

## Requirements

- Elementor (free or Pro)
- jQuery
