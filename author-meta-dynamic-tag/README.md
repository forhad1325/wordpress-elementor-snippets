# Elementor Author Meta Link Dynamic Tag

Adds a custom **Author Meta Link** dynamic tag to Elementor. Reads any user meta field from the post author and returns it as a URL or mailto link.

## Usage

1. Paste the code into your theme's `functions.php` or a custom plugin.
2. In Elementor, use the **Author Meta Link** dynamic tag on any URL/link field.
3. Enter the meta key (e.g. `linkedin_url`, `work_mail`) in the tag settings.

## Requirements

- Elementor Pro
